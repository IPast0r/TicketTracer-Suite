Welcome to TicketTracer

Quick start guide

This platform is designed to streamline IT support by tracking incidents (generating a tickets) and organizing them in categories. Once created you can edit or delete them completely.

1 - Manage Categories

Start by generating your service areas (Hardware, Software, O.S ...) in the categories section.

2 - Manage a Ticket

Use the ticket section on the sidebar to report a new incident. Assign it a category, a priority, a distinctive name and a detailed description. Once the incident is resolved close it by using the delete action.
